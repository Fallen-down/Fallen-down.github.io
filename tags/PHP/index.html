Ajax 跨域 一起来吧！

Ajax 简介

Ajax读音：[ˈeɪdʒæks]

Ajax 的全程：Asynchronous JavaScript and XML（异步的 JavaScript 和 XML）。
Ajax 不是某种编程语言，是一种在无需重新加载整个页面的情况之下能够更新网页部分的技术。

解决问题

传统的网页（不使用 Ajax 技术的页面），想要更新内容或者提交一个表单，就需要重新载入页面。
使用了 Ajax 技术的页面，通过在后台跟服务器进行少量的数据交互，网页就可以实现异步局部更新。
