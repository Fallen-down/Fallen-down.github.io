CSS 单位

em 根据父元素的字体大小变化。

rem 根据根元素的字体大小变化。

vw 显示窗口的宽度
vh 显示窗口的高度
vmin 和 vmax 宽度和高度的最大值或最小值有关（如果宽度为1100px高度为700px 那么 1vmin 是 7px，1vmax 是 11px 反之亦然。）

ex和ch单位 这两个单位只也依赖于font-family，因为它们被定为基于特殊字体的法案。
